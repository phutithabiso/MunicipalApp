# MunicipalApp

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Overview
**MunicipalApp** is a desktop application that allows residents to manage municipal services efficiently. Users can submit and track service requests, manage local events, and generate reports. The application integrates advanced data structures for optimal performance and visualization.

---

## Features

- **Service Request Management**
  - Add, update, and view service requests
  - Track request statuses
  - Search requests by ID or filter by criteria

- **Local Event Management**
  - Add and view municipal events
  - Track event details

- **Reports**
  - Generate and view service request and event reports

- **Data Structures Integration**
  - Binary Search Tree (BST)
  - AVL Tree
  - MinHeap
  - Graph structures for visualization

- **User Interface**
  - Interactive Windows Forms GUI
  - Visual representation of requests using TreeView and ListBox
